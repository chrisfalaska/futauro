# Say hello to Auro!

The open-source design system elevating your ideas.

## Philosophy

Enabling design and development to innovate ideas and collaborate on shaping the future.

## Getting Started

Install the design system in your project:

```bash
npm install @auro
```

Use components in your HTML:

```html
<auro-button>Click me</auro-button>
```

Or import them in JavaScript:

```javascript
import '@auro/button';
```

## Core Principles

- **Consistency**: All components follow our design guidelines
- **Accessibility**: Built with WCAG 2.1 guidelines in mind
- **Performance**: Optimized for runtime performance
- **Maintainability**: Well-documented and easy to update
